# Fire Alerting System using ESP32

This project is a simple fire alerting system built using ESP32 and sensors.
When fire or abnormal conditions are detected, the system sends an alert message to a Telegram Bot and can trigger a local alarm.

# Technologies Used

* ESP32
* Flame/Smoke Sensor
* Telegram Bot
* Arduino IDE

# My Role

This project was developed as part of a college project.
I set up the hardware using ESP32 and sensors, configured the Telegram bot for alert notifications, and tested the system to ensure it works correctly.

The code used in this project was adapted from online resources and AI assistance and modified to work with the hardware setup.
